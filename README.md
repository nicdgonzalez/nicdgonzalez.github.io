# Portfolio Website

My personal portfolio website, built with [Astro].

## Installation

Clone the repository:

```bash
git clone https://github.com/nicdgonzalez/nicdgonzalez.github.io
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser to <http://127.0.0.1:4321>.

[astro]: https://astro.build/
